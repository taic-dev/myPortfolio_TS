## Description
Taic-devのポートフォリオサイト。<br>
エンジニアであれば誰しもが1度は目を通したことがある公式ドキュメント。<br>
公式ドキュメントを参考にシンプルなデザインでコンテンツを見やすく、わかりやすくまとめることを意識して制作しました。<br>
また、爆速でコンテンツが表示されるように、サイトの構築にはNext.jsを使用しユーザーにストレスを与えないサイトを目指しました。
